# The Classification of Potholes from Motion Sensor Data

This MSc Dissertation project develops a machine learning pipeline to detect potholes and road surface anomalies using smartphone accelerometer and gyroscope data. It aimed to provide a scalable, low-cost solution for automated road condition monitoring.

The workflow includes preprocessing raw time-series sensor data, extracting features using sliding window techniques, and training supervised models such as Random Forest and SVM for classification. Signal processing methods are applied to handle noise and improve feature quality.

The model is evaluated using standard classification metrics, with visualisations used to explore patterns in sensor signals and assess performance.
